Documentation

This plugin is really easy to use. After you download the plugin add the following lines between the HEAD tag on your page:

    <link rel="stylesheet" href="sigslider.min.css">
    <script type="text/javascript" src="jquery.min.js"></script>
    <script type="text/javascript" src="jquery.sexyslider.min.js"></script>

Next all you have to do is create a wrapper with some ID and put the next code below it:

The above code will take all images from the div 'wrapper' and will create the slider. If you want to comment below images just add text in alt tag.

    <div id='wrapper'>
      <img src='image1.jpg' />
      <img src='image2.jpg' alt='description' />
      <img src='image3.jpg' />
    </div>

Things you can customize: (With their default values)

    transitions   : 'random',       // #id of div to place control buttons.
    width         : 768,            // width of slider
    height        : 300,            // height of slider
    boxCols       : 20,             // number of column strips
    boxRows       : 10,             // number of row strips
    animSpeed     : 1000,           // Animation speed
    pauseTime     : 3000,           // delay between images in ms
    startSlide    : 1,              // start slider with the Index of the image
    directionNav  : true,           // hide or show direction controller
    controlNav    : true,           // hide or show control controller
    pauseOnHover  : true,           // on mouse over pause the transistion of next image
    loader        : 'loader.png',   // Loader image path

Programmatically set transition

    $('#slider').data("sigaslider").setTransition("ShrinkCol");

Programmatically trigger next and previous slides

    $('#slider').data("sigaslider").next();
    $('#slider').data("sigaslider").prev();

Programmatically set image slide

    $('#slider').data("sigaslider").setSlide(2);

Example

HTML:

    <div id='wrapper'>
      <img src='image1.jpg' alt='#captionId' />
      <img src='image2.jpg' alt='description' />
      <img src='image3.jpg' />
    </div>

JS:

    <script type="text/javascript">
      $(document).ready(function(){
        $('#slider').SigaSlider({
          width  : 768,
          height : 300,
          boxCols : 20,
          boxRows  : 10,
          animSpeed  : 1000,
          pauseTime  : 3000,
          transitions : '['BounceUp', 'Dizzle']' //"random",
        });
      });
    </script>
